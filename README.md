## Description

This project is a GUI-based Electronic Cash Machine (ATM) developed using C# and Windows Forms.  
It simulates real ATM operations such as PIN login, balance enquiry, withdrawal, and receipt generation.

---

## Features

- 4-digit PIN authentication
- Balance enquiry
- Withdraw money
- Receipt display
- Insufficient funds protection
- Dynamic account storage using List<Account>
- Inheritance (SavingsAccount and CurrentAccount)
- Polymorphism
